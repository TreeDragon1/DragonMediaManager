# Long-Term Vision

Dragon Media Centre will become the single application used to manage and enjoy an entire home media system.

Jellyfin, Sonarr, Radarr, Prowlarr, Bazarr, qBittorrent and Docker remain the powerful services underneath, while Dragon Media Centre becomes the one interface users interact with every day.
