# 🐉 Dragon Media Centre

> **Your Personal Media Centre**

Dragon Media Centre is a modern desktop application for managing and, in the future, enjoying your complete home media ecosystem from one place.

Instead of opening multiple applications, Dragon Media Centre brings everything together into one interface.

---

## ✨ Current Features

- 🎬 Jellyfin Integration
- 📺 Sonarr Integration
- 🎞️ Radarr Integration
- 🔍 Prowlarr Integration
- 💬 Bazarr Integration
- ⬇️ qBittorrent Integration
- 🐳 Docker Health Monitoring
- 💾 Automatic Backups
- 🤖 Dragon AI
- 🩺 Dragon Health Dashboard
- ⚡ Quick Actions
- 📊 Media Statistics

## 🚀 Installation

```bash
git clone https://github.com/TreeDragon1/DragonMediaManager.git

cd DragonMediaManager

python3 -m venv .venv

source .venv/bin/activate

pip install -r requirements.txt

python app.py
```

## 🎯 Vision

Dragon Media Centre aims to become one application that can:

- Manage your media server
- Monitor system health
- Control Docker services
- Browse Movies
- Browse TV Shows
- Play media
- Run on Linux, Windows, Android, Android TV and the Web
